기본 열람 권한
업로드 시 설정한 기본 열람 권한은 문서 등록 후 보안 및 권한 제어 화면에서 세부 수정할 수 있습니다.
전체 공개 문서는 모든 직원이 챗봇/RAG 검색 결과로 접근할 수 있습니다.
관리자 전용 문서는 관리자 권한 사용자만 챗봇/RAG 검색 결과로 접근할 수 있습니다.
조건 조합
부서, 역할, 직책, 등급 조건을 조합하여 기본 접근 대상을 지정합니다. 예: 개발본부 + 선임 이상
선택한 조건은 AND 조건으로 적용됩니다. 예를 들어 대상 부서가 개발본부이고 직책 기준이 선임 이상이면, 개발본부 소속이면서 선임 이상인 직원만 해당 문서에 접근할 수 있습니다.
